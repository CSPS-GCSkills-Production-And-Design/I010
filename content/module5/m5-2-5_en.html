Adding documents and emails to GCDOCS

Try It

Try adding a document to GCDOCS using Enterprise Connect.

  • Save the document: Budget Meeting Minutes.doc. to the Financial Management Folder in the Enterprise Workspace.
  • Name the document and describe the keywords as: technology, budget meeting, minutes.

To follow the steps, select the Step by Step button below.

To start the simulation, select the Play button and, then select the Start button.

A hint will be displayed if you select any location on the screen.
