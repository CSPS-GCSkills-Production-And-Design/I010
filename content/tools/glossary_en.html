Glossary

  • Advanced search

    An advanced search in GCDOCS for complex search criteria (for example: dates, synonyms and more). You can include all GCDOCS item types in your search or restrict it to certain types.

  • Attribute

    An attribute is a piece of metadata that describes an item. GCDOCS automatically generates system attributes (such as date created and created by) for each item added to the system. These attributes enable you, for example, to search for items created by a specific person on a certain date.

  • Browse

    Browsing is the act of viewing items within a container, such as a folder or workspace.
    For example, you browse the Enterprise Workspace to see its contents.

  • Collection

    A collection allows you to group information together, and then share with your co-workers

  • Document

    An electronic file, in any format, that is stored in GCDOCS. GCDOCS stores, audits, controls access to, and maintains information pertaining to documents.

  • Edit

    Edit is the act of modifying an item, such as a folder or document.

  • Email link

    The email link option allows you to quickly share items in GCDOCS by sending a link by email.

  • Enterprise Connect

    Enterprise Connect integrates GCDOCS with the applications you use on a daily basis, including MS Word, PowerPoint, Excel and Outlook.

  • Enterprise Workspace

    The location in GCDOCS which serves as a shared repository for business related information.

  • Favourites

    The My Favourites page is the user’s personal space where one can organize and access important documents and items of interest. Adding an item as a favorite sets it apart from other items in GCDOCS, thereby making it easily accessible.

  • Folder

    A folder is a container that can store many items, such as documents, emails, and tasks.

  • Folder Structure

    Provides you with a way to organize and maintain your files and folders through a classification method.

  • Functions Menu

    A menu of functions that appears when you click an item's Functions icon. Only the functions you have permission to perform appear on this menu.

  • GCDOCS

    Is the government-wide solution for storing, accessing, and managing business information, and is part of the Open Government initiative.

  • Item

    An item is anything that you are able to create or add in GCDOCS, such as documents, folders and URL’s.

  • Keywords

    Keywords are words or phrases that you specify to search for items. For example, if you specify the keywords "business plan model" as your search criteria, GCDOCS returns a list of items that contain all three words, any of these words, or the exact phrase.

  • Last Modified Date

    Allows you to search based on the date a document (or its profile) was last modified.

  • Locked Version

    A locked version is a version of a document that cannot be deleted.

  • My Profile

    The relevant information defining a GCDOCS user, including general information that is defined when the user is created and personal information added by the user.

  • My Workspace

    My workspace is a personal storage area that only you, by default, can access.

  • Natural Language Query

    A natural language query allows you to use one or more sentences or sentence fragments when searching. For example, if your natural language query is "Where can I find a presentation about collaboration software?”, GCDOCS returns a list of items that contain each or some of the following words: "presentation", "collaboration", and "software".

  • Notification

    Notifications are a feature that notifies you of changes in GCDOCS, such as the addition of a document to a folder.

  • Notification Report

    The messages generated by GCDOCS notifications according to certain settings. Each user can customize three report settings for personal use.

  • Open

    Open displays the document in its native application, such as MS Word. Using open allows you to view a document but edit it.

  • Permissions

    Permissions are rules that define what GCDOCS users and groups can do to a particular item or within a certain location.

  • Personal menu

    A work area and list of important links for each user. The menu includes a link to the user’s Personal Workspace (i.e., a folder that only users can access). The personal menu also contains links to pages of information the user is most likely to use.

  • Personal settings

    Personal settings allow you to set your personal preferences for layout and content, appearances, and the workspace.

  • Quick Search

    The Quick Search provides a simplified approach to locating your requirements in the system.

  • Relevance

    The score that GCDOCS has assigned to the result based on how closely the result matches your search criteria.

  • Rename

    Rename is the act of changing the name of an item for which you have the modify permission. Renaming an item does not interfere with any shortcuts or links that reference that item.

  • Reserve

    Reserve is the act of write-locking a GCDOCS item, preventing other users from making any changes.

  • Search Result

    A search result is a list of GCDOCS items returned when you submit a search. All items in a search result meet the criteria specified in your search. For example, if you search for "vacation policy", your search results page will list all GCDOCS items that contain the phrase "vacation policy".

  • Set Notification

    Setting a notification is the act of defining settings to notify you of changes in GCDOCS.

  • Shortcut

    A shortcut is a pointer to an item stored elsewhere in GCDOCS. You can create multiple shortcuts to an item in different locations. This provides quick and easy access to that item. A shortcut always refers to the latest Version of the original item.

  • Slice

    A slice is a portion of an area in GCDOCS that you can search. You can use slices to narrow the scope of your searches.

  • Unreserve

    Unreserve is the act of cancelling the write-lock on a GCDOCS item, allowing other users to reserve or modify the item.

  • URL

    A GCDOCS item containing a Uniform Resource Locator (URL), known as an Internet address.

  • Version Control

    Version control is the act of keeping multiple versions of an item stored in GCDOCS. Version control allows you to set the number of versions you want to keep, purge older versions of an item, and lock versions.

  • User Help

    Describes and explains how to use every feature and function in GCDOCS, and also describes in detail how they work and what they are designed to do.

  • Version

    A version in GCDOCS captures and preserves an existing item and its modifications. GCDOCS allows you to preserve and maintain multiple versions of items.

  • Wiki

    A wiki is a website which allows its users to add, modify, or delete its content via a web browser. Wikis serve many different purposes, such as knowledge management, collaboration, and community engagement.
