Contrôle des versions

Quand un utilisateur modifie un document, une nouvelle version de ce document est créée automatiquement. Dans GCDOCS, le contrôle des versions vous permet de suivre plusieurs versions d'un même document et, par conséquent, d’en préserver l'évolution. Remarque : Il n'y a qu'un utilisateur à la fois qui peut modifier un document dans GCDOCS.

Le contrôle des versions indique le nom de la personne qui a créé la version, le moment auquel elle l'a fait ainsi que le numéro attribué à la version. GCDOCS vous permet de verrouiller certaines versions de façon à ce qu'elles ne puissent pas être supprimées et d'éliminer des versions antérieures qui ne sont plus utiles d'un point de vue opérationnel.
