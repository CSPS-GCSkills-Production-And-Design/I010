Emailing document links

Sending a link

When sharing your work using email, it's always preferable to send a document link rather than an attachment. Whereas an attachment is a copy of the document; a link gives the recipient access to the latest version of the document. Thus sending a link eliminates the confusion around which document is the latest version. Sending the link also ensures that only users with sufficient permissions can access or modify the document. When recipients of the email select the link, they are brought to the Document Properties page. Depending on their level of permissions, they can either open the document in a read-only format or edit the document in its native application, such as MS Word or PowerPoint.
