Concepts et compétences clés

Félicitations! Vous avez terminé tous les modules du Cours de base sur GCDOCS : En ligne – À rythme libre. Vous êtes maintenant prêt(e) à commencer à utiliser le système!

Voici quelques concepts présentés dans ce module qu’il est important de se rappeler.

Gestion des versions

  • GCDOCS vous permet de suivre les différentes versions d’un document et de déterminer qui l’a créé et à quel moment, et quel est le numéro de la version. Vous pouvez également verrouiller certaines versions de façon à ce qu’elles ne puissent être supprimées ou éliminer les versions antérieures.
    • Comment voir les différentes versions d’un document
    • Comment verrouiller un document
    • Comment contrôler les versions

Envoi par courriel de liens de documents

  • Il est toujours préférable d’envoyer un lien de document plutôt qu’une pièce jointe afin de s’assurer que les destinataires ont la version la plus récente.
  • En envoyant un lien, vous vous assurez également que seuls les utilisateurs qui possèdent les autorisations appropriées peuvent consulter ou modifier le document. Vous n’avez plus à craindre qu’une pièce jointe ne soit pas envoyée aux bonnes personnes.
