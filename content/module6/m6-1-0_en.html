Managing versions

Version control

When a user modifies a document, a new version of that document is created automatically. In GCDOCS, version control allows you to track several versions of a documentâ€”ensuring that the evolution of a document is preserved. Note: Only one user at a time can modify a document in GCDOCS.

Versioning tracks who created the version, when it was created, and the version number. GCDOCS allows you to lock certain versions so that they cannot be deleted or purge older versions that no longer have any business value.
