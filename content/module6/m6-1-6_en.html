Managing versions

Adjusting versioning

GCDOCS also allows you to select the type of versioning that will be applied to documents. There are two kinds of versioning:

Standard versioning applies versions in a linear format. Version 1 will be followed by version 2, which will be followed by version 3, and so on.

Advanced versioning applies major and minor versions. Major versions are represented by 1.0, 2.0, 3.0; while minor versions are represented as 0.1, 0.2, 0.3, and so on.

The advantage of Advanced versioning is that it allows you to promote certain versions to a “major version.” Since users with "See Contents" permissions can only view major versions, this is a great way to manage access to specific versions.
