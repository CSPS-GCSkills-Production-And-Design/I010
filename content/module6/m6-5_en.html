Congratulations! You've completed all modules of GCDOCS Fundamentals: Online Self-paced and are ready to get started using the system.

Here are some important concepts to remember from this module.

Managing versions

  • GCDOCS allows you to track the versions of a document and determine: who created it, when it was created, and the version number. You can also Lock certain versions so they can't be deleted or purge older versions.
    • How to view versions of a document
    • How to lock a document
    • How to adjust versioning

Emailing document links

  • It is always preferable to send a document link rather than an attachment. This helps ensure that recipients always have the latest version.
  • Sending a link also ensures that only users with sufficient permissions can access or modify the document. No more worrying about email attachments being forwarded to the wrong individuals.
