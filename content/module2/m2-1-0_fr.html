Lancer GCDOCS

Ouverture d’une session

GCDOCS est un outil Web que vous pouvez utiliser au moyen du navigateur Internet de votre ordinateur au travail. Pour lancer GCDOCS, saisir l’adresse Web ou l’URL fournie par votre organisation dans la barre d’adresse Web.

Vous arriverez alors à la page d’accueil de GCDOCS.

  • Pour apprendre comment ouvrir une session, cliquez sur le lien suivant pour une démonstration : Comment ouvrir une session
  • Pour essayer vous-même, cliquez sur : Essayez-le
