Aperçu : Comment utiliser GCDOCS

Dans ce module, vous vous familiariserez avec le système GCDOCS. Vous apprendrez comment utiliser le système et vous vous exercerez à naviguer à certaines zones essentielles.

À la fin du module, vous serez en mesure de faire ce qui suit :

  • ouvrir une session GCDOCS
  • reconnaître les zones essentielles de l’interface
  • naviguer à travers ces zones
  • régler certains paramètres personnels, par exemple la page de démarrage par défaut
  • fermer la session GCDOCS
