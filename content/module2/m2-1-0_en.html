Launching GCDOCS

Logging-in

GCDOCS is a web-based tool that can be accessed through your desktop internet browser at work. To launch GCDOCS, type in the web address or URL provided by your organization in the web address bar.

This will take you to the GCDOCS log-in page.

  • To learn how to log-in, select the following link for a demonstration: How to log-in
  • To try it yourself, select: Try it
