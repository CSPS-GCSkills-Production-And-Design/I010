Réglage des paramètres personnels

Changer la page de démarrage par défaut

Vous connaissez maintenant les zones essentielles de l’interface GCDOCS. Essayons alors de réaliser une tâche élémentaire pour amorcer votre utilisation du système.

Dans ce volet, vous allez apprendre comment régler vos paramètres personnels. La fonction Paramètres de GCDOCS vous permet de personnaliser certaines des caractéristiques du système. Vous pouvez, par exemple, changer la palette de couleurs ou la langue de l’interface de GCDOCS, ou encore changer la page de démarrage par défaut.

  • Pour découvrir comment régler les paramètres personnels afin de changer votre page de démarrage par défaut, choisir : Comment changer la page de démarrage par défaut
  • Pour l’essayer vous-même, cliquez sur : Essayez-le
