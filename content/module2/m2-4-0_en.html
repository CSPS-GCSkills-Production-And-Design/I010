Adjusting personal settings

Changing the default start page

Now that you know the key areas of the GCDOCS interface, let's try an easy task to get you started using the system.

In this section you learn how to adjust your personal settings. The Settings function in GCDOCS allows you to personalize some of the system's features. For example, you can change the colour scheme or language of the GCDOCS interface or change your default start page.

  • To learn how to adjust personal settings to change your default start page, select: How to change the default start page
  • To try it yourself, select: Try it
