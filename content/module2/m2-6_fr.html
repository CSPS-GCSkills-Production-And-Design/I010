Concepts et compétences clés

Ce module vous a permis de vous familiariser avec le système GCDOCS. Voici une synthèse de quelques éléments dont vous avez besoin pour commencer :

Comment lancer GCDOCS

  • Sur la page d’accueil, saisir votre nom d’utilisateur et votre mot de passe et, par la suite, cliquer sur Ouvrir une session. Vous serez alors dirigé vers votre page de démarrage par défaut.

Zones essentielles de l’interface

  • Barre de menu : pour avoir accès à votre espace de travail Entreprise, votre espace de travail personnel et le menu Outils.
  • Bouton aide : pour obtenir de l’aide sur toutes les fonctions.
  • Filtre de contenu : pour faire le tri des éléments affichés à l’écran.
  • Pistes de navigation : pour déterminer votre emplacement actuel.
  • Outils de dossiers et de documents : pour avoir accès aux dossiers et aux documents et pouvoir les manipuler.
  • Recherche : pour trouver rapidement les éléments dont vous avez besoin.

Comment régler vos paramètres personnels

  • La fonction Paramètres de GCDOCS vous permet de personnaliser certaines caractéristiques du système. Vous pouvez, par exemple, changer la palette de couleurs, la langue d’utilisation, ou la page de démarrage par défaut. L’option Paramètres se trouve dans le menu Outils.

Comment fermer une session

  • Fermer une session en choisissant Déconnexion du menu Outils ou en fermant votre navigateur.
