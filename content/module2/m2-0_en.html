Overview: Getting Started with GCDOCS

In this module, you will become acquainted with the GCDOCS system. You will learn how to get started using the system and practice navigating to some key areas.

After completing the module, you will be able to:

  • log-in to GCDOCS
  • identify key areas of the interface
  • navigate to key areas
  • adjust personal settings like the default start page
  • log-out of GCDOCS
