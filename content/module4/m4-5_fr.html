Concepts et compétences clés

Dans ce module, vous avez découvert la capacité de recherche de GCDOCS. Voici quelques concepts importants à retenir :

Effectuer une recherche rapide

  • La fonction Recherche rapide est disponible en tout temps. Vous n’avez qu’à saisir vos mots-clés dans le champ de recherche situé dans le coin supérieur droit de l’écran et cliquer ensuite sur l’icône Effectuer la recherche pour lancer votre recherche.
  • La fonction Recherche effectue la recherche à l’endroit où vous vous trouvez dans le système. Par exemple, si vous travaillez dans un dossier de l’espace de travail Entreprise, GCDOCS n’effectuera la recherche que dans ce dossier.
  • Vous pouvez utiliser des opérateurs comme « et », « ou », ou « non ».

Affiner une recherche

  • Vous pouvez affiner davantage votre recherche en cliquant sur l’icône Ouvrir le panneau de recherche.
  • Au moyen des options du Panneau de recherche, vous pouvez préciser :
    • Vues de recherche : l’endroit où vous souhaitez chercher;
    • Type d’objet : le type de document que vous cherchez, en fonction de l’application dans laquelle il a été créé;
    • Créé par : personne qui a créé le document;
    • Date de dernière modification : la date de la dernière modification du document.
  • Si vous cliquez sur À partir d’ici dans le menu Vues de recherche, GCDOCS lance la recherche à partir du dossier qui est affiché.

Interpréter les résultats de recherche

  • Après avoir effectué une recherche, une page de résultats de recherche s’affiche à l’écran, présentant les documents correspondant à vos critères de recherche.
  • Le taux de pertinence, exprimé en pourcentage (%), indique dans quelle mesure le document est pertinent en fonction des mots-clés ou des critères de recherche. Plus le pourcentage est élevé, plus il est susceptible de correspondre à vos critères de recherche.
  • Le menu Sélectionner l’action situé sur les pages des résultats de recherche vous donne des options pour gérer les résultats de recherche
    (p. ex.,: ajouter les résultats de sa recherche à une « collection », ou encore appliquer une « attente »).
  • La fonction Enregistrez la requête de recherche (accessible par le menu Outils de recherche) vous permet d’enregistrer vos résultats de recherche.

Accéder à un document

  • Lorsque vous choisissez un élément des résultats de recherche, la page Aperçu du document s’affiche.
  • La page Aperçu du document vous donne les options suivantes :
    • Ouvrir une copie du document en lecture seule;
    • Envoyer un lien au document par courriel qui crée un message courriel contenant un lien à l’élément choisi;
    • Modifier le document dans son application d’origine (uniquement pour les utilisateurs ayant la permission de modifier des documents).
