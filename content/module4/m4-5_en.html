Key concepts and skills

In this module you learned about the search capability of GCDOCS. Here are some key concepts to remember:

Conducting a Quick Search

  • The Quick Search function is always available to use. Just enter your keywords into the Search field at the top right of the screen and press the perform search icon to conduct your search.
  • The Search function searches the location where you currently are in the system. For example, if you are working in a folder in the Enterprise Workspace, GCDOCS will search just that folder.
  • You can use operators like "and," "or," or "not."

Refining a Search

  • You can further refine your search by selecting the Open Search Panel icon.
  • Using the Search Panel options you can specify:
    • Slices: where you want to search,
    • Object type: the type of document you are searching for based on the application it was created in,
    • Created by: who created the document, and
    • Last Modified: the date it was last edited or updated.
  • If you select From Here in the Slices menu, GCDOCS begins the search from the folder you are currently displaying.

Interpreting Search Results

  • After conducting a search, a Search Results page is displayed with the documents matching your query.
  • The Relevance score, expressed as a percentage (%), indicates how relevant a document is to your keywords or search criteria. The higher the percentage, the more likely it matches your search criteria.
  • The Select Action menu on the Search Results pages gives you options for managing search results (e.g., add to "collections" or apply a "hold").
  • The Save Search Query function (accessed through the Search Tools menu), allows you to save your search.

Accessing a Document

  • When you select an item from Search Results, the document's Overview Page is displayed.
  • The Overview Page give you options to:
    • Open a read-only copy of the document
    • E-mail document link which creates an email message with a link to the selected object, or
    • Edit the document in its native application (only available to users with Modify permissions).
