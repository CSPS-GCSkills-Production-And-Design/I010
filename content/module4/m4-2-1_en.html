Interpreting Search Results

Marc's Search Results (continued)

Two other important features on the Search Results page allow Marc to efficiently manage the documents he has found.

1. Select Action menu. This drop-down menu gives Marc access to various document management functions. For example, he can add his search results to a collection.

2. Search Tools menu. Using the Save Search Query functions in this menu, Marc can save his search results. This will save time and effort by allowing him to quickly repeat the same search or a similar one in the future.
