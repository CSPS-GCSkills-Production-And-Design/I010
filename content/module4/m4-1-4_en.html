Conducting a Quick Search

Refining your search

You can further refine your search by opening the Open Search Panel icon beside the search field. The expanded Search Panel is shown here.

Slices allows you to indicate where in GCDOCS you want to search. From Here will begin the search from the folder you are currently displaying. Pressing the downward arrow will expand the Slices menu and give you other options.

Created by allows you to refine your search based on who created the document (e.g. anyone, created by me, etc.). Pressing the downward arrow will expand the Object type menu and display the options.

Object Type allows you to define the type of document you are searching for based on the application it was created in (e.g., MS Word, Excel, etc.) Pressing the downward arrow will expand the Object type menu and display the options.

Last Modified allows you to search based on the date a document was last edited or updated.
