Effectuer une recherche rapide

Affiner votre recherche

Vous pouvez affiner davantage votre recherche en cliquant sur l’icône Ouvrir le panneau de recherche, située à côté du champ de recherche. Le panneau de recherche étendu est présenté ici.

Vues de recherche vous permet d’indiquer dans quel endroit de GCDOCS vous souhaitez chercher. À partir d’ici amorcera la recherche dans le dossier affiché à l’écran. Si vous cliquez sur la flèche pointant vers le bas, le menu des vues de recherche s’affichera et vous verrez d’autres options.

Type d’objet vous permet de préciser le genre de document que vous cherchez en fonction de l’application dans laquelle il a été créé (p. ex. : MS Word, Excel, etc.). Si vous cliquez sur la flèche pointant vers le bas, le menu Type d’objet s’affichera et vous verrez des options.

Créé par vous permet d’affiner votre recherche en fonction de la personne qui a créé le document (p. ex. : toute personne, créé par moi, etc.). En cliquant sur la flèche vers le bas, vous verrez s’afficher les options du menu Type d’objet.

Date de dernière modification vous permet de faire une recherche en fonction de la date de la dernière modification du document.

  • Pour l’essayez vous-même, cliquez sur : Essayez-le
