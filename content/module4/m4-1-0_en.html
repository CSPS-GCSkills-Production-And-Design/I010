Marc's challenge

Marc wants to provide his team with some tips for time management. He remembers reading and saving a document that included some excellent tips in the appendix, but he can't remember the title of the document or its author.

Select the function on the screen that Marc would use to find the document using keywords such as "time" and "management."

To follow the steps, select the Step by Step button below.

To start the simulation, select the Play button and, then select the Start button.

A hint will be displayed if you select any location on the screen.
